## Usage

### Kbd

| Prop        | `color`                                                                                                                                                                                                                                                                          |
| ----------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Description | The kbd's color.                                                                                                                                                                                                                                                                 |
| type        | `'brand' \| 'slate' \| 'gray' \| 'zinc' \| 'neutral' \| 'stone' \| 'red' \| 'orange' \| 'amber' \| 'yellow' \| 'lime' \| 'green' \| 'emerald' \| 'teal' \| 'cyan' \| 'sky' \| 'blue' \| 'indigo' \| 'violet' \| 'purple' \| 'fuchsia' \| 'pink' \| 'rose' \| 'white-black'{:ts}` |
| default     | `'neutral'{:ts}`                                                                                                                                                                                                                                                                 |
| required    | `false{:ts}`                                                                                                                                                                                                                                                                     |

| Prop        | `variant`                                     |
| ----------- | --------------------------------------------- |
| Description | The kbd's style.                              |
| type        | `'outline' \| 'plane' \| 'skeuomorphic'{:ts}` |
| default     | `'outline'{:ts}`                              |
| required    | `false{:ts}`                                  |
