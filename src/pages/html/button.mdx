## Usage

### Solid

| Prop        | `color`                                                                                                                                                                                                                                                                          |
| ----------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Description | The button's color.                                                                                                                                                                                                                                                              |
| type        | `'brand' \| 'slate' \| 'gray' \| 'zinc' \| 'neutral' \| 'stone' \| 'red' \| 'orange' \| 'amber' \| 'yellow' \| 'lime' \| 'green' \| 'emerald' \| 'teal' \| 'cyan' \| 'sky' \| 'blue' \| 'indigo' \| 'violet' \| 'purple' \| 'fuchsia' \| 'pink' \| 'rose' \| 'white-black'{:ts}` |
| default     | `'blue'{:ts}`                                                                                                                                                                                                                                                                    |
| required    | `false{:ts}`                                                                                                                                                                                                                                                                     |

| Prop        | `size`                                                                                 |
| ----------- | -------------------------------------------------------------------------------------- |
| Description | The button's size.                                                                     |
| type        | `'small' \| 'medium' \| 'large' \| 'icon-small' \| 'icon-medium' \| 'icon-large'{:ts}` |
| default     | `'medium'{:ts}`                                                                        |
| required    | `false{:ts}`                                                                           |

| Prop        | `variant`                                |
| ----------- | ---------------------------------------- |
| Description | The button's style.                      |
| type        | `'default' \| 'outline' \| 'ghost'{:ts}` |
| default     | `'default'{:ts}`                         |
| required    | `false{:ts}`                             |
