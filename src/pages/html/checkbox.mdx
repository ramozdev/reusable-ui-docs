### TailwindCSS Forms Plugin

You must install the [@tailwindcss/forms](https://github.com/tailwindlabs/tailwindcss-forms) to use this component.

<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tab>
    
    ```bash copy
    pnpm install -D @tailwindcss/forms
    ```

  </Tab>
  <Tab>
    
    ```bash copy
    npm install -D @tailwindcss/forms
    ```

  </Tab>
  <Tab>
    
    ```bash copy
    yarn add -D @tailwindcss/forms
    ```

  </Tab>
</Tabs>

Then add the plugin to your `tailwind.config.ts` file:

```ts copy filename="tailwind.config.ts" {5}
import type { Config } from "tailwindcss";

const config: Config = {
  // ...
  plugins: [require("@tailwindcss/forms")],
};

module.exports = config;
```

## Usage

### Checkbox

| Prop        | `color`                                                                                                                                                                                                                                                                          |
| ----------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Description | The checkbox's color.                                                                                                                                                                                                                                                            |
| type        | `'brand' \| 'slate' \| 'gray' \| 'zinc' \| 'neutral' \| 'stone' \| 'red' \| 'orange' \| 'amber' \| 'yellow' \| 'lime' \| 'green' \| 'emerald' \| 'teal' \| 'cyan' \| 'sky' \| 'blue' \| 'indigo' \| 'violet' \| 'purple' \| 'fuchsia' \| 'pink' \| 'rose' \| 'white-black'{:ts}` |
| default     | `'blue'{:ts}`                                                                                                                                                                                                                                                                    |
| required    | `false{:ts}`                                                                                                                                                                                                                                                                     |

| Prop        | `variant`                   |
| ----------- | --------------------------- |
| Description | The checkbox's style.       |
| type        | `'outline' \| 'plane'{:ts}` |
| default     | `'outline'{:ts}`            |
| required    | `false{:ts}`                |
