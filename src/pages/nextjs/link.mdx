# Link

Client-side transitions between routes can be enabled via the `Link` component exported by `next/link`.

<NextjsDocs slug="link" />

## Underline

<Tabs items={['Demo', 'Code']}>
  <Tab>
  </Tab>
  <Tab>
   
```tsx copy
import * as Link from "@/ui/nextjs/link";
import { colors } from "@/lib/colors";

export function Default() {
  return (
    <div className="flex flex-wrap gap-4">
      {colors.map((color) => (
        <Link.Underline
          key={`link-solid-underline-${color}`}
          href="#underline"
          color={color}
        >
          {color}
        </Link.Underline>
      ))}
    </div>
  );
}
```

## Usage

### Solid

| Prop        | `color`                                                                                                                                                                                                                                                                          |
| ----------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Description | The link's color.                                                                                                                                                                                                                                                                |
| type        | `'brand' \| 'slate' \| 'gray' \| 'zinc' \| 'neutral' \| 'stone' \| 'red' \| 'orange' \| 'amber' \| 'yellow' \| 'lime' \| 'green' \| 'emerald' \| 'teal' \| 'cyan' \| 'sky' \| 'blue' \| 'indigo' \| 'violet' \| 'purple' \| 'fuchsia' \| 'pink' \| 'rose' \| 'white-black'{:ts}` |
| default     | `'blue'{:ts}`                                                                                                                                                                                                                                                                    |
| required    | `false{:ts}`                                                                                                                                                                                                                                                                     |

| Prop        | `size`                                                                                 |
| ----------- | -------------------------------------------------------------------------------------- |
| Description | The link's size.                                                                       |
| type        | `'small' \| 'medium' \| 'large' \| 'icon-small' \| 'icon-medium' \| 'icon-large'{:ts}` |
| default     | `'medium'{:ts}`                                                                        |
| required    | `false{:ts}`                                                                           |

| Prop        | `variant`                                |
| ----------- | ---------------------------------------- |
| Description | The link's style.                        |
| type        | `'default' \| 'outline' \| 'ghost'{:ts}` |
| default     | `'default'{:ts}`                         |
| required    | `false{:ts}`                             |
